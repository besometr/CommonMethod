КонструкторСтруктурыОбъекта

Конструктор структуры всех реквизитов и табличных частей объекта по полному имени метаданных.

 ПараметрОбязательныйТипОписание
1ПолноеИмяОбъектаМетаданныхДаСтрокаНаименование метаданных, по которым будет сформировано описание объекта.
2СтруктураРеквизитовДляПоискаНетСтруктураНеобязательный. В качестве ключа передается имена реквизитов,
а в качестве значения (опционально) - значение реквизита.
3ТекстОшибкиНетСтрокаСодержит описание ошибки, в случае возникновения оной.
4ОтказНетБулевоПризнак возникновения ошибки выполнения.

Возвращаемое значение

Структура - Описание объекта. В качестве ключа передаются имена реквизитов, а в качестве значения - значение реквизита.
Реквизиты с именами табличных частей заполнены пустыми массивами.
Неопределено - Если не удалось получить объект метаданных по имени или найденный вид метаданных не поддерживается конструктором.

или найденный объект не поддерживается подсистемой "Общие методы объектов".

КонструкторОбъекта

Конструктор Объекта по полному имени метаданных.

 ПараметрОбязательныйТипОписание
1ПолноеИмяОбъектаМетаданныхДаСтрокаНаименование метаданных, по которым будет осуществляться поиск табличной части.
2СтруктураОбъектаДаСтруктураСодержит реквизитов и табличные части объекта,
Для реквизитов в качестве ключа передаются имена реквизитов, а в качестве значения - значение реквизита.
Для табличных частей в качестве ключа передаются имена табличных частей, а в качестве значения - массивы структур реквизитов ТЧ.
3ДополнительныеСвойстваНетСтруктураЗначения для заполнения одноименной структуры объекта.
Используется в случаях, когда необходимо хранить некоторые значения, связанные с объектом,
на время выполнения некоторых операций, без изменения объекта.
4ПараметрыЗаписиНетСтруктураПараметры для записи документов (Режим Записи и Режим Проведения).
Если не заполненно, используются параметры по умолчанию конкретного Объекта.
5ТекстОшибкиНетСтрокаСодержит описание ошибки, в случае возникновения оной.
6ОтказНетБулевоПризнак возникновения ошибки выполнения.

Возвращаемое значение

Ссылка - Ссылка на созданный объект.
Неопределено - В случае ошибки.

ОбновитьОбъект

Выполняет вызов функции изменения Объекта по полному имени метаданных.

 ПараметрОбязательныйТипОписание
1СсылкаДаСсылкаСсылка на изменяемый объект.
2СтруктураИзмененийОбъектаДаСтруктураСодержит реквизиты и табличные части объекта, которые нужно изменить.
Для реквизитов в качестве ключа передаются имена реквизитов, а в качестве значения - значение реквизита.
Для табличных частей в качестве ключа передаются имена табличных частей, а в качестве значения - массивы структур реквизитов ТЧ.
3ДополнительныеСвойстваНетСтруктураНеобязательный. Значения для заполнения одноименной структуры объекта.
Используется в случаях, когда необходимо хранить некоторые значения, связанные с объектом,
на время выполнения некоторых операций, без изменения объекта.
4ПараметрыЗаписиНетСтруктураНеобязательный. Параметры для записи документов (Режим Записи и Режим Проведения).
5ОчищатьТЧНетБулевоНеобязательный. Если Истина, перед добавлением строк в указанные в Структуре Источнике табличные части,
последние будут очищены.
6ТекстОшибкиНетСтрокаСодержит описание ошибки, в случае возникновения оной.
7ОтказНетБулевоПризнак возникновения ошибки выполнения.

Возвращаемое значение

Ссылка - Ссылка на измененный объект.
Неопределено - В случае ошибки записи.

КонструкторПараметровЗаписи

Конструктор структуры параметров записи для документов по полному имени метаданных.

 ПараметрОбязательныйТипОписание
1ПолноеИмяОбъектаМетаданныхДаСтрокаНаименование метаданных, для которых будет осуществляться поиск Параметров записи по умолчанию.
2РежимЗаписиНетРежимЗаписиДокументаНеобязательный. Позволяет выбрать один из возможных режимов записи.
Значение по умолчанию: Запись.
3РежимПроведенияНетРежимПроведенияДокументаНеобязательный. Позволяет выбрать один из возможных режимов проведения.
Значение по умолчанию: Неоперативный.
4ТекстОшибкиНетСтрокаСодержит описание ошибки, в случае возникновения оной.
5ОтказНетБулевоПризнак возникновения ошибки выполнения.

Возвращаемое значение

Структура - Параметры для записи документа. Если по полному наименованию найден объект метаданных,
будет возвращена структура параметров записи по умолчанию конкретного объекта.
В противном случае общая структура по умолчанию.
